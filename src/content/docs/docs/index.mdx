Welcome to the BmSDK docs. BmSDK is a C#-based script hook and modding SDK for
**Batman: Arkham City** and **Batman: Arkham Knight**. It hooks into Unreal
Engine 3's event loop and exposes a full C# SDK so you can extend the games with
custom gameplay and logic.
